# make-project
## まとめてセットアップ
```sh
make setup
```

## 個別にセットアップ(インストールは別途必要)
### dotfiles
```sh
make setup-dotfiles
```

### zsh
```sh
make setup-zsh
```

### neovim
```sh
make setup-neovim
```

### rust
```sh
make setup-rust
```

### install pyenv goenv nodenv by using anyenv
```sh
make setup-env
```

### poetry
```sh
make setup-poetry
```

### alacritty
```sh
make setup-alacritty
```
